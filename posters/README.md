# Posters Folder

Tải hình poster của phim vào folder này.

**Ví dụ:**
- `conan.jpg`
- `dune.jpg`
- `oppenheimer.png`

Cách sử dụng:
1. Trong trang Admin, click chọn file poster
2. Hệ thống sẽ tự động lưu poster dưới dạng data URL (trong localStorage)
3. Hoặc bạn có thể tải file vào folder này và nhập URL: `/posters/tên-file.jpg`
